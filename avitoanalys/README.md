# Парсер и анализ объявлений Авито

Парсит объявления с маркетплейса Авито, анализирует цены, распределение по регионам и находит тренды.

## Что делает
- Парсит объявления в Telegram (поиск по названию товара)
- Собирает название, цену, город, ссылку
- Очищает и нормализует данные
- Анализирует распределение цен
- Находит топ дорогих и дешёвых товаров
- Группирует по городам/регионам

## Результаты (пример: iPhone 13)
- Спарсено: 100 объявлений
- Средняя цена: 23404 ₽
- Диапазон: 16490 - 31490 ₽
- Анализ по районам Москвы

## Технологии
Python, Playwright, BeautifulSoup4, pandas, numpy
