# Парсер книжного каталога

Собирает название и цену книг с сайта-каталога (books.toscrape.com), 
проходит по нескольким страницам автоматически и сохраняет результат в Excel.

## Что умеет
- Обходит пагинацию (заданное количество страниц)
- Обрабатывает ошибки сети (таймаут, недоступность сервера)
- Сохраняет результат в готовый Excel-файл

## Технологии
Python, requests, BeautifulSoup4, pandas, openpyxl

## Как запустить
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить: `python parsing.py`
