# Парсер цитат (JS-версия)

Парсит цитаты с сайта, где контент рендерится через JavaScript 
(используется Playwright для открытия настоящего браузера).

Собирает текст цитаты, автора и теги для каждой цитаты, сохраняет в Excel.

## Что умеет
- Открывает страницу через Playwright (запускает настоящий браузер)
- Ждёт, пока контент загрузится (wait_for_selector)
- Собирает данные после полной загрузки страницы (контент через JS)
- Сохраняет результат в Excel с несколькими листами

## Технологии
Python, Playwright, BeautifulSoup4, pandas

## Как запустить
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить: `python playwrightpars.py`
3. Результат в `quotes_js.xlsx`
