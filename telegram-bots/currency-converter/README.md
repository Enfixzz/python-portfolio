# Конвертер валют в Telegram

Простой Telegram-бот для конвертации иностранных валют в российские рубли.

## Что умеет
- Конвертировать доллары (USD), евро (EUR) и фунты (GBP) в рубли
- Интерактивное меню с inline-кнопками
- Мгновенный расчёт курса

## Технологии
Python, aiogram 3.x, asyncio
