# Бот регистрации пользователей

Telegram-бот для регистрации пользователей с пошаговым диалогом и сохранением данных в БД.

## Что умеет
- Пошаговая регистрация: запрос имени и возраста
- Подтверждение введённых данных через кнопки
- Сохранение пользователей в SQLite-базу
- Запоминание зарегистрированных пользователей (повторное приветствие)
- Валидация возраста (только числа)
- Возможность переregистрации при ошибке

## Технологии
Python, aiogram 3.x, SQLAlchemy ORM, SQLite, FSM (Finite State Machine)
