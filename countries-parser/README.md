# Парсер стран: анализ плотности населения

Собирает данные о странах мира (название, столица, население, площадь) 
с сайта-каталога и вычисляет плотность населения для каждой страны. 
Сохраняет топ-15 самых плотнонаселённых стран в CSV-файл.

## Что умеет
- Парсит полный список стран одним запросом (без пагинации)
- Вычисляет производную метрику (плотность населения = население / площадь), 
  которой нет в исходных данных на сайте
- Обрабатывает некорректные или отсутствующие данные без падения скрипта
- Сортирует страны по плотности населения и сохраняет топ-N в CSV

## Технологии
Python, requests, BeautifulSoup4

## Как запустить
1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить: `python bs4pars.py`
